    Changelog

    2021-02-08 Черный список

    • Внесены изменения в структуру документации DataAPI
    • Добавили работу с Черным списком

    Метод Описание
    "get.blacklist_phone_numbers" Получение телефонных номеров занесенных в черный список
    "delete.phone_number_from_blacklist" Удаление номера телефона из черного списка
    "create.phone_number_to_blacklist" Добавление номера телефона в черный список

    2019-11-14

    Добавили в "Загрузку коммуникаций"

    • Загрузка/удаление офлайн-заявок upload.offline_messages
    • Загрузка/удаление чатов upload.chats

    [2.0.9] - 2019-10-03

    Добавили

    • Получение аккаунта get.account

    [2.0.8] - 2019-01-10

    Добавили

    • Создание сайта create.sites
    • Удаление сайта delete.sites
    • Редактирование сайта update.sites

    [2.0.7] - 2018-11-29

    Добавили

    • Получение статистики РК по дням. Метод get.campaign_daily_stat
    • Получение списка виртуальных номеров доступных для подключения get.available_virtual_numbers"
    • Подключение виртуального номера enable.virtual_numbers
    • Отключение виртуального номера disable.virtual_numbers
    • Добавили параметр include_ongoing_calls "Включить текущие вызовы" для метода get.calls_report

    [2.0.6] - 2018-08-10

    Добавили

    • Добавили в методы create.employees, get.employees, update.employees параметры управления статусом сотрудника и разрешениями на связь: status, allowed_in_call_types, allowed_out_call_types;
    • Параметр calls_available будет удален из методов create.employees, get.employees, update.employees через 2-а месяца. Параметр не рекомендуется к использованию.

    [2.0.5] - 2018-07-12

    Добавили

    • Добавили параметр время обработки потерянных звонков "lost_call_processing_duration" в метод get.calls_report;
    • Добавили новый метод получения списка сценариев get.scenarios;
    • Добавили новый метод получения медиафайлов - системные, пользовательские get.media_files.

    [2.0.4] - 2018-02-15

    Добавили

    • Добавили возможность получения списка сайтов. Метод get.sites;
    • Добавили возможность получения списка блоков номеров сайта. Метод get.site_blocks;
    • В методах создания сотрудника и группы сотрудников максимальная длина внутреннего номера расширена до 5 знаков;
    • Добавили возможность управления SIP-линиями. Методы get.sip_lines, create.sip_lines, update.sip_lines, delete.sip_lines, update.sip_line_password.

    [2.0.3] - 2018-02-07

    Добавили

    • Добавили возможность получения списка подключенных виртуальных номеров. Метод get.virtual_numbers;
    • В метод get.goals_report добавили информацию об интегрированных целях. Параметры type, source, ext_id;
    • В метод get.calls_report добавили регион номера абонента. Параметр cpn_region_id, cpn_region_name;
    • В метод get.call_legs_report добавили информацию о группе в рамках которой был звонок, а так же информацию об операции в рамках которой звонок был создан. Параметры group_id, group_name, action_id, action_name.

    [2.0.2] - 2018-01-29

    Добавили

    • Методы управления рекламными кампаниями - создание, удаление, получение, редактирование;
    • В метод "Управление отчетами -> get.visitor_sessions_report" добавили информацию об интегрированных рекламных кампаниях. Параметр integrated_campaign_data;
    • В методе get.calls_report расширили допустимое значение параметра finish_reason.
