    Генерация нового пароля для SIP-линии

    SIP-линии > Генерация нового пароля для SIP-линии
    Метод update.sip_line_password
    Описание Генерация нового пароля для SIP-линии
    Кому доступен Партнёр, Клиент

    Параметры запроса

    Название Тип Обязательный Описание
    access_token string да Ключ сессии аутентификации
    id number да Уникальный идентификатор SIP-линии
    user_id number нет Уникальный идентификатор пользователя клиента агента от имени которого делается запрос
    Является обязательным для агента
    Для получения списка пользователей клиента необходимо использовать метод "get.customer_users"

    Параметры ответа

    Название Тип Описание
    password string Новый пароль для регистрации

    JSON структура запроса

    {
  "jsonrpc": "2.0",
  "id": "number",
  "method": "update.sip_line_password",
  "params": {
    "access_token": "string",
    "user_id": "number",
    "id": "number"
  }
}

    JSON структура ответа

    {
  "jsonrpc": "2.0",
  "id": "number",
  "result": {
    "password": "string"
  }
}
