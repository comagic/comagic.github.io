    Редактирование пула номеров

    Сайты > Редактирование пула номеров
    Метод update.number_pool
    Описание Редактирование пула номеров
    Кому доступен Агент, Клиент

    Параметры запроса

    Название Тип Обязательный Допустимые значения Описание
    access_token string да Ключ сессии аутентификации
    user_id number нет Уникальный идентификатор пользователя клиента агента от имени которого делается запрос
    Является обязательным для агента
    Для получения списка пользователей клиента необходимо использовать метод "get.customer_users"
    id number да Уникальный идентификатор пула номеров.
    name string да Название пула номеров
    description string нет Описание пула номеров
    state enum да active, draft Состояние пула номеров
    phone_id number нет Уникальный идентификатор cтатического номера. Используется только, если is_online = false
    redirection_phone string нет Номер для переадресации.Обязателен и заполняется только, если is_online = false В запросе может быть либо redirection_phone либо call_processing_rule_group_id
    call_processing_rule_group_id number нет Группа правил обработки звонков.Обязателен и заполняется только, если is_online = false В запросе может быть либо redirection_phone либо call_processing_rule_group_id
    traffic_type enum нет none, all, custom_sources, conditions Вид трафика. Если traffic_type принимает значение: * none или all, то массивы conditions и ac_ids пустые * custom_sources, то массив conditions пустой, а ac_ids не пустой * conditions, то массив conditions не пустой
    number_pool_template_id number нет Уникальный идентификатор шаблона условий пула
    Условия шаблона пула
    conditions object нет Список условий шаблона пула
    operator string Значение оператора между группами условий
    Массив условий в группе условий
    values array Условия рекламной кампании в группе условий
    operator string Значение оператора между условиями внутри группы
    values array Условия рекламной кампании в группе условий
    parameter enum Список допустимых значений и возможных операторов см. в таблице "Список параметров и их операторов"
    values array Массив значений для parameter.
    Для некоторых parameter может быть свой список допустимых значений см. в таблице "Список параметров и их операторов"
    operator enum =, !=, like, not_like, ~ Оператор в условии
    • "=" - точно соответствует
    • "!=" - точно не соответствует
    • "like" - строго включает
    • "not_like" - строго исключает
    • "~" - соответсвтует регулярному выражение
    ac_ids array нет Массив рекламных кампаний
    Метки для звонков. Вы можете добавить дополнительные UTM-метки, которые будут всегда заполнены у звонков по данному пулу.
    eq_utms array нет Метки для звонков. Вы можете добавить дополнительные UTM-метки, которые будут всегда заполнены у звонков по данному пулу.
    eq_utm enum нет Список допустимых значений см. в Простовляемая на звонок метка
    value string нет Значение простовляемой на звонок метки
    Настройки групп телефонных номеров для замены на номера коллтрекинга
    replacement_phones_group_settings array нет Настройки групп телефонных номеров для замены на номера коллтрекинга. Используется если is_online = true
    replacement_phones_group_id number нет Идентификатор группы телефонных номеров для замены на номера коллтрекинга. Метод получение списка существующих групп: "get.replacement_phones_groups"
    is_dynamic boolean нет ture, false Использовать динамический коллтрекинг
    phone_id number нет Уникальный идентификатор cтатического номера. Обязателен и заполняется только, если is_dynamic = false
    prefix string нет Префиск номеров коллтрекинга. Обязателен и заполняется только, если is_dynamic = true
    virtual_numbers_count number нет Минимальное значение = 2 Общее количество номеров коллтрекинга в одном блоке. Необходимое количество номеров будет докуплено. Обязателен и заполняется только, если is_dynamic = true
    visitors_count number нет Минимальное значение = 1 Количество посещений в сутки. Обязателен и заполняется только, если is_dynamic = true
    call_processing_rule_group_id number нет Группа правил обработки звонков. В запросе может быть либо redirection_phone либо call_processing_rule_group_id
    redirection_phone string нет Номер для переадресации. В запросе может быть либо redirection_phone либо call_processing_rule_group_id
    coverage_visitors number нет Прогноз охвата. Фактические данные по переходам посетителей еще не накоплены – рассчитывается прогноз. Охват зависит от времени резервирования, количества динамических номеров и посещаемости в сутки.

  "jsonrpc": "2.0",
  "id": "number",
  "method": "update.number_pool",
  "params": {
    "access_token": "string",
    "user_id": "number",
    "id": "number",
    "name": "string",
    "description": "string",
    "phone_id": "number",
    "state": "enum",
    "number_pool_template_id": "number",
    "conditions": 
        {
          "operator": "string",
          "values": [
            {  
              "operator": "string",
              "values": [
                {
                  "parameter": "enum",
                  "operator": "enum",
                  "values": [
                    "string"
              ]
            }
          ]
        }
      ]
    },
    "ac_ids": [
      value1,
      value2,
      ...
    ],
    "eq_utms":[
          {
            "eq_utm":"enum",
            "value":"string"
          }
    ],
    "call_processing_rule_group_id":"number",
    "redirection_phone": "string",    
    "replacement_phones_group_settings":[
          {
            "replacement_phones_group_id": "number",
            "is_dynamic": "boolean",
            "phone_id": "number",
            "prefix": "string",
            "virtual_numbers_count": "number",
            "visitors_count": "number",
            "coverage_visitors": "number",
            "call_processing_rule_group_id":"number",
            "redirection_phone": "string"            
          }
        ]
      }
}
