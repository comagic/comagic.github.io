    Редактирование пула номеров

    Сайты > Редактирование ПКТ
    Метод update.call_tracking_rule
    Описание Редактирование ПКТ
    Кому доступен Агент, Клиент

    Параметры запроса

    Название Тип Обязательный Допустимые значения Описание
    access_token string да Ключ сессии аутентификации
    user_id number нет Уникальный идентификатор пользователя клиента агента от имени которого делается запрос
    Является обязательным для агента
    Для получения списка пользователей клиента необходимо использовать метод "get.customer_users"
    id number да Уникальный идентификатор ПКТ
    name string нет Название ПКТ
    is_named_by_user boolean Обязательно, если передан параметр name Признак, что название введено ручную
    rule_type enum нет site_calls, external_area Вид ПКТ. site_calls - Звонки с сайтов. external_area - Звонки со сторонних площадок. manual_settings - Ручная настройка.
    reserve_phone_numbers_time number да Время резерва динамического номера за посетителем в минутах (от 5 до 10080)
    call_tracking_type enum нет static, dynamic Тип коллтрекинга. Для rule_type = external_area, допустимо только значение static
    state enum нет active, draft Состояние ПКТ
    is_all_traffic boolean нет ture, false Использовать данное ПКТ для всего траффика
    sources array нет Список источников, для которых используется данное ПКТ. Может присутствовать в запросе только если call_tracking_type = dynamic
    source_channel_id number да Идентификатор канала источника. Метод получение списка существующих каналов: "get.get_sources_channels"
    source_group_id number нет Идентификатор группы источника. Метод получение списка существующих групп: "get.get_sources_groups"
    source_id number нет Идентификатор источника. Метод получение списка существующих Источников: "get.get_sources"
    conditions array нет Условия для определения ПКТ, допустимо только для rule_type = manual_settings
    filters array Выражение, может содержать в себе как простые фильтры, так и дерево фильтров
    field string Список допустимых значений и возможных операторов см. в таблице "Список параметров и их операторов
    operator enum Список допустимых значений и возможных операторов см. в таблице "Список параметров и их операторов"
    value string Значение для оператора фильтрации.
    external_replacement_settings array нет Настройка статических номеров. Может присутствовать в запросе только если call_tracking_type = static и rule_type = site_calls
    id number нет Идентификатор блока с настройкой.
    replacement_phones_groups array да Идентификатор группы телефонных номеров и сайтов для замены на номера коллтрекинга.
    id number да Идентификатор группы телефонных номеров. Метод получение списка существующих групп: "get.replacement_phones_groups"
    site_id number да Идентификатор сайта.
    source_id number нет Идентификатор источника. Метод получение списка существующих Источников: "get.get_sources"
    static_phone_id number да Номер для статического коллтрекинга.
    child_customer_id number Идентификатор клиента агента, для которого заведен сайт. Метод получение списка клиентов: "get.customers"
    call_processing_rule_group_id number нет Одно из значений
