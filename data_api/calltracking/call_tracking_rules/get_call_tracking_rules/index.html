    Получение всех ПКТ.

    Сайты > Получение всех пулов номеров.
    Метод get.call_tracking_rules
    Описание Получение всех ПКТ.
    Кому доступен Агент, Клиент

    Параметры запроса

    Название Тип Обязательный Допустимые значения Описание
    access_token string да Ключ сессии аутентификации
    user_id number нет Уникальный идентификатор пользователя клиента агента от имени которого делается запрос
    Является обязательным для агента
    Для получения списка пользователей клиента необходимо использовать метод "get.customer_users"
    limit number нет Количество возвращаемых записей. См. раздел "Постраничный вывод"
    offset number нет Сдвиг, определяет с какого номера записи возвращать `"limit"` записей. По умолчанию "0". См. раздел "Постраничный вывод"
    search_string string нет Поискова строка, фильтрующая данные в ответе
    fields array нет См. раздел ["Представление возвращаемых данных"](../index.md#_views)
    Фильтры применяемые в отчете
    filter object нет Допустимо использовать не более 2 уровней вложенности. Фильтры применяемые в отчете. Логика построения аналогична "Критерии фильтрации"
    filters array да Выражение, может содержать в себе как простые фильтры, так и дерево фильтров. Доступный список фильтров для отчета с их возможными значениями см.раздел "Получение списка возможных условий фильтрации для отчета" .
    field string Поле сущности к которой будет применяться фильтрация (список заранее определён для метода)
    operator enum Оператор фильтрации. Список всех операторов можно получить в разделе "Операторы фильтрации"
    value string Значение для оператора фильтрации. Необязательное поле, если оно отсутствует, то считается пустота.
    Сортировка применяемая в отчете
    sort array нет Сортировка применяемая в отчете. Логика построения аналогична "Сортировка данных"

    Параметры ответа

    Название Тип Допустимые значения Фильтрация Сортировка Описание
    id number да да Уникальный идентификатор ПКТ
    name string да да Название ПКТ
    is_named_by_user boolean да да Признак, что название введено ручную
    priority string нет нет Приоритет
    rule_type enum site_calls, external_area нет нет Вид ПКТ
    reserve_phone_numbers_time number да Время резерва динамического номера за посетителем в минутах (от 5 до 10080)
    Условия для определения ПКТ.
    conditions array нет Условия для определения ПКТ, допустимо только для call_tracking_type = dynamic, is_fraffic_type = false, и пустом sources
    filters array Выражение, может содержать в себе как простые фильтры, так и дерево фильтров
    field string Список допустимых значений и возможных операторов см. в таблице "Список параметров и их операторов
    operator enum Список допустимых значений и возможных операторов см. в таблице "Список параметров и их операторов"
    value string Значение для оператора фильтрации.
    call_tracking_type enum static, dynamic нет нет Тип коллтрекинга
    site_ids array да нет Идентификаторы сайтов, на которые настроен с ПКТ
    child_customer_ids array да нет Идентификаторы клиентов агента связанные с ПКТ
    Список сайтов связанных с ПКТ.
    sites array нет нет Сайты связанные с ПКТ
    id number нет нет Идентификатор сайта
    name string нет нет Название
    domain string нет нет Домен
    Список источников, на которые настроено ПКТ.
    sources string нет нет Источники, которые связаны с ПКТ
    source_channel_id string нет нет Идентификатор канала
    source_group_id string нет нет Идентификатор группы
    source_id string нет нет Идентификатор источника
    name string нет нет Название источника или группы или канала
    is_available_edit boolean нет нет Признак допустимости редактирования данного ПКТ данным пользователем
    is_available_set_priority boolean нет нет Признак допустимости редактирования приоритета
    state enum active, draft нет нет Статус ПКТ
    is_all_traffic boolean нет нет Признак настройки ПКТ на весь трафик
    coverage number Минимальный фактический охват по всем пулам ПКТ
    is_multi_redirections number Признак настроенной переадрисации на разные номера и правила ВАТС
    redirection_numbers number Номер переадресации ПКТ, если он единственный
    call_processing_rule_groups number Группа правил ВАТС для переадресации ПКТ, если она единственная
    Настройки переадресации для звонков со сторонних площадок.
    external_replacement_settings array Настройка статических номеров. Может присутствовать в запросе только если call_tracking_type = static и rule_type = site_calls
    id number Идентификатор блока с настройкой.
    replacement_phones_groups array Идентификатор группы телефонных номеров и сайтов для замены на номера коллтрекинга.
    id number Идентификатор группы телефонных номеров. Метод получение списка существующих групп: "get.replacement_phones_groups"
    site_id number Идентификатор сайта.
    source_id number Идентификатор источника. Метод получение списка существующих Источников: "get.get_sources"
    static_phone_id number Номер для статического коллтрекинга.
    child_customer_id number Клиент агента
    Группа правил переадресации.
    call_processing_rule_group number Группа правил обработки звонков. В запросе может быть либо custom_redirection_phone либо redirection_phone_id либо call_processing_rule_group
    id number Идентификатор группы
    name string Название группы
    custom_redirection_phone string Номер для переадресации. запросе может быть либо custom_redirection_phone либо redirection_phone_id либо call_processing_rule_group
    redirection_phone_id number Номер для переадресации. запросе может быть либо custom_redirection_phone либо redirection_phone_id либо call_processing_rule_group
    Настройки статических номеров и переадресации звонков.
    static_replacement_phones_settings array Настройка статических номеров. Может присутствовать в запросе только если call_tracking_type = static и rule_type = site_calls
    id number Идентификатор блока с настройкой.
    replacement_phones_groups array Идентификатор группы телефонных номеров и сайтов для замены на номера коллтрекинга.
    id number Идентификатор группы телефонных номеров. Метод получение списка существующих групп: "get.replacement_phones_groups"
    site_id number Идентификатор сайта.
    source object Источник, на которые настроена гарантия.
    source_channel_id number Идентификатор канала источника. Метод получение списка существующих каналов: "get.get_sources_channels"
    source_group_id number Идентификатор группы источника. Метод получение списка существующих групп: "get.get_sources_groups"
    source_id number Идентификатор источника. Метод получение списка существующих Источников: "get.get_sources"
    name string нет нет Название источника или группы или канала
    default_source number Источник для отображения в отчете. Метод получение списка существующих Источников: "get.get_sources"
    source_channel_id string нет нет Идентификатор канала
    source_group_id string нет нет Идентификатор группы
    source_id string нет нет Идентификатор источника
    name string нет нет Название источника
    static_phone_id number Номер для статического коллтрекинга.
    call_processing_rule_group number Группа правил обработки звонков. В запросе может быть либо custom_redirection_phone либо redirection_phone_id либо call_processing_rule_group
    id number Идентификатор группы
    name string Название группы
    custom_redirection_phone string нет Номер для переадресации. запросе может быть либо custom_redirection_phone либо redirection_phone_id либо call_processing_rule_group
    redirection_phone_id number Номер для переадресации. запросе может быть либо custom_redirection_phone либо redirection_phone_id либо call_processing_rule_group
    Настройки динамической автоматического выбора номеров и правила обработки звонков.
    dynamic_replacement_phones_settings array Настройки групп телефонных номеров для замены на номера динамического коллтрекинга. Может присутствовать в запросе только если call_tracking_type = dynamic
    id number Идентификатор блока с настройкой.
    replacement_phones_groups array Идентификатор группы телефонных номеров и сайтов для замены на номера коллтрекинга.
    id number Идентификатор группы телефонных номеров. Метод получение списка существующих групп: "get.replacement_phones_groups"
    site_id number Идентификатор сайта.
    prefix string Префиск номеров коллтрекинга. Обязателен и заполняется только, если is_dynamic = true
    virtual_numbers_count number Общее количество номеров коллтрекинга в одном блоке. Необходимое количество номеров будет докуплено.
    visitors_count number Количество посещений в сутки. Обязателен и заполняется только, если is_dynamic = true
    call_processing_rule_group number Группа правил обработки звонков. В запросе может быть либо custom_redirection_phone либо redirection_phone_id либо call_processing_rule_group
    id number Идентификатор группы

    redirection_phone_id number Номер для переадресации. запросе может быть либо custom_redirection_phone либо redirection_phone_id либо call_processing_rule_group
    Настройки гарантии на источник.
    sources_bundle_settings array Гарантия на источник, позволяет сохранить источник при звонке на статические номера. Может присутствовать в запросе только если call_tracking_type = dynamic
    id number Идентификатор блока с настройкой.

    id number Идентификатор группы телефонных номеров. Метод получение списка существующих групп: "get.replacement_phones_groups"
    site_id number Идентификатор сайта.
    source object Список источников, на которые настроена гарантия.

    source_group_id number Идентификатор группы источника. Метод получение списка существующих групп: "get.get_sources_groups"
    source_id number Идентификатор источника. Метод получение списка существующих Источников: "get.get_sources"
