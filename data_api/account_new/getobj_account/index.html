    Получение информации об аккаунте

    Получение информации об аккаунте > Получение информации об аккаунте
    Метод getobj.account
    Описание Получение информации об аккаунте
    Кому доступен Агент, Клиент
    Необходимые компоненты для работы с методом Data API для настройки
    Вес метода 1 балл

    Параметры запроса

    Название Тип Обязательный Описание
    access_token string да Ключ сессии аутентификации
    user_id number нет Уникальный идентификатор пользователя клиента агента от имени которого делается запрос
    Является обязательным для агента
    Для получения списка пользователей клиента необходимо использовать метод "get.customer_users"

    Параметры ответа

    Название Тип Допустимые значения Описание
    app_id number Уникальный идентификатор аккаунта
    app_state string Состояние аккаунта
    app_name string Имя аккаунта
    is_agent_app boolean Является ли клиентом агента
    customer_id number Уникальный идентификатор аккаунта во внутренней CRM
    user_id number Уникальный идентификатор пользователя аккаунта
    user_type string Тип пользователя аккаунта
    user_login string Логин пользователя аккаунта
    user_name string Имя пользователя аккаунта
    tp_id number Идентификатор текущего тарифного плана аккаунта
    tp_name string Название текущего тарифного плана аккаунта
    crm_type string Тип подключенной CRM системы
    lang string Язык интерфейса аккаунта
    project string Тип проекта аккаунта
    timezone string Формат IANA zoneinfo Текущий часовой пояс аккаунта
    permissions array Права и уровни доступа
    unit_id string Мнемоника доступа
    is_delete boolean Есть ли доступ на удаление
    is_insert boolean Есть ли доступ на создание
    is_select boolean Есть ли доступ на чтение
    is_update boolean Есть ли доступ на редактирование
    components array Список компонентов, подключенных у клиента
    is_bi_available boolean Показывает добавлен ли клиент в OLAP
    report_currency string Валюта используемая в отчетах
    is_agent boolean Является ли аккаунт агентским
    agent_type enum
    • agent
    • franchisor
    		 Тип агента

    JSON структура запроса

    {
  "jsonrpc":"2.0",
  "id":"number",
  "method":"getobj.account",
  "params":{
    "access_token":"string"
  }
}
