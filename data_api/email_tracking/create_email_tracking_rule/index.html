    Создание правил email трекинга

    Метод create.email_tracking_rule
    Описание Создание правил email трекинга
    Кому доступен Агент, Клиент

    Параметры запроса

    Название Тип Обязательный Допустимые значения Описание
    access_token string да Ключ сессии аутентификации
    user_id number нет Уникальный идентификатор пользователя клиента агента от имени которого делается запрос
    Является обязательным для агента
    Для получения списка пользователей клиента необходимо использовать метод "get.customer_users"
    name string да Название правила.
    description string нет Описание правила.
    rule_type string да
    • site_emails - Подмена на сайте
    • external_area - Email из сторонних площадок
    		 Тип правила
    traffic_type text да
    • all - Весь трафик
    • custom_sources - Пользовательская настройка трафика
    		 Тип трафика для email-tracking
    tracking_type text да static, dynamic Тип email трекинга
    substitution_type string Допустим и обязателен только для tracking_type = dynamic.
    • auto_generation - Автоматическая генерация 10000 почт
    • manual_pool - Ручной ввод почт
    		 Способ формирования пула Email для подмены
    state text да
    • active - Активное правило
    • draft - Черновик
    		 Статус правила email трекинга
    sources array Обязательно tracking_type = dynamic и traffic_type = custom_sources, иначе отсутсвует Список источников, для которых используется данное правило.
    source_channel_id number да Идентификатор канала источника. Метод получение списка существующих каналов: "get.get_sources_channels"
    source_group_id number нет Идентификатор группы источника. Метод получение списка существующих групп: "get.get_sources_groups"
    source_id number нет Идентификатор источника. Метод получение списка существующих Источников: "get.get_sources"
    external_replacement_emails_settings array Может присутствовать в запросе только если tracking_type = static И rule_type = external_area Настройка статических Email на сторонних площадках.
    email_account_id number да Email адрес для отображения на сайте: "get.email_accounts".
    child_customer_id number Идентификатор клиента, для которого заведен почтовый ящик. Метод получение списка клиентов: "get.customers"
    source object нет Список источников, на которые настроена гарантия. В случае если нужно выбрать Весь трафик, в данный параметр передается null
    source_channel_id number да Идентификатор канала источника. Метод получение списка существующих каналов: "get.get_sources_channels"
    source_group_id number нет Идентификатор группы источника. Метод получение списка существующих групп: "get.get_sources_groups"
    source_id number нет Идентификатор источника. Метод получение списка существующих Источников: "get.get_sources"
    static_replacement_emails_settings array Может присутствовать в запросе только если tracking_type = static И rule_type = site_emails Настройка статических Email на сайте
    replacement_emails_groups array Обязателен для rule_type = site_emails Группы email и сайтов для замены на подменяемые email.
    id number да Идентификатор группы email. Метод получение списка существующих групп: "get.replacement_emails_groups"
    site_id number да Идентификатор сайта.
    email_account_id number да Email адрес для отображения на сайте: "get.email_accounts".
    source object нет Список источников, на которые настроен блок. В случае если нужно выбрать Весь трафик, в данный параметр передается null
    source_channel_id number да Идентификатор канала источника. Метод получение списка существующих каналов: "get.get_sources_channels"
    source_group_id number нет Идентификатор группы источника. Метод получение списка существующих групп: "get.get_sources_groups"
    source_id number нет Идентификатор источника. Метод получение списка существующих Источников: "get.get_sources"
    default_source_id number Обязательно при rule_type = site_emails Идентификатор источника для отображения в отчете. Метод получение списка существующих Источников: "get.get_sources"
    dynamic_replacement_emails_settings array Может присутствовать в запросе только если tracking_type = dynamic Настройка динамических Email.
    replacement_emails_groups array Обязателен для rule_type = site_emails Группы email и сайтов для замены на подменяемые email.
    id number да Идентификатор группы email. Метод получение списка существующих групп: "get.replacement_emails_groups"
    site_id number да Идентификатор сайта.
    email_account_id number да Email адрес для отображения на сайте: "get.email_accounts".
    email_accounts array Допустим и обязателен при substitution_type = manual_pool Список почт для подмены в случае ручного набора почт.
