    Список правил email трекинга

    Метод get.email_tracking_rules
    Описание Список правил email трекинга
    Кому доступен Агент, Клиент

    Параметры запроса

    Название Тип Обязательный Допустимые значения Описание
    access_token string да Ключ сессии аутентификации
    user_id number нет Уникальный идентификатор пользователя клиента агента от имени которого делается запрос
    Является обязательным для агента
    Для получения списка пользователей клиента необходимо использовать метод "get.customer_users"
    limit number нет Количество возвращаемых записей. См. раздел "Постраничный вывод"
    offset number нет Сдвиг, определяет с какого номера записи возвращать `"limit"` записей. По умолчанию "0". См. раздел "Постраничный вывод"
    search_string string нет Поискова строка, фильтрующая данные в ответе
    fields array нет См. раздел ["Представление возвращаемых данных"](../index.md#_views)
    Фильтры применяемые в отчете
    filter object нет Допустимо использовать не более 2 уровней вложенности. Фильтры применяемые в отчете. Логика построения аналогична "Критерии фильтрации"
    filters array да Выражение, может содержать в себе как простые фильтры, так и дерево фильтров. Доступный список фильтров для отчета с их возможными значениями см.раздел "Получение списка возможных условий фильтрации для отчета" .
    field string Поле сущности к которой будет применяться фильтрация (список заранее определён для метода)
    operator enum Оператор фильтрации. Список всех операторов можно получить в разделе "Операторы фильтрации"
    value string Значение для оператора фильтрации. Необязательное поле, если оно отсутствует, то считается пустота.
    Сортировка применяемая в отчете
    sort array нет Сортировка применяемая в отчете. Логика построения аналогична "Сортировка данных"

    Параметры ответа

    Название Тип Допустимые значения Фильтрация Сортировка Описание
    id number да да Идентификатор правила
    name string нет нет Название правила
    description string нет нет Описание правила
    priority number да да Приоритет правила
    creation_date date да да Дата создания правила
    tracking_type string
    • static - Статический;
    • dynamic - Динамический
    		 да да Тип email-tracking
    substitution_type string
    • auto_generation - Автоматическая генерация 10000 почт
    • manual_pool - Ручной ввод почт
    		 да да Способ формирования пула Email для подмены
    traffic_type string
    • all - Весь трафик
    • custom_sources - Пользовательская настройка трафика
    		 да да Тип трафика для email-tracking
    rule_type string
    • site_emails - Подмена на сайте
    • external_area - Email из сторонних площадок
    		 да да Тип правила
    state string
    • active - Активное правило
    • draft - Черновик
    		 нет нет Состояние
    is_available_edit boolean нет нет Доступно ли редактирование
    is_available_set_priority boolean нет нет Доступно ли пользователю редактировать приоритет
    site_ids array да нет Идентификаторы сайтов, на которые настроен с ИМТ
    child_customer_ids array да нет Идентификаторы клиента агента, связанные с ИМТ
    Список сайтов связанных с ИМТ.
    sites array нет нет Сайты связанные с ПКТ
    id number нет нет Идентификатор сайта
    name string нет нет Название
    domain string нет нет Домен
    Настройки переадресации для email со сторонних площадок.
    external_replacement_emails_settings array Может присутствовать в запросе только если tracking_type = static И rule_type = external_area нет нет Настройка статических Email на сторонних площадках.
    id number Идентификатор блока с настройкой.
    email_account object нет нет Адрес электронной почты
    id number нет нет ИД адреса электронной почты
    name string нет нет Название адреса электронной почты
    child_customer_id number Идентификатор клиента, для которого заведен сайт. Метод получение списка клиентов: "get.customers" /td>
    sources array нет нет Источники для динамической подмены
    source_channel_id number нет нет Канал
    source_group_id number нет нет Источник
    source_id number нет нет Источник детально
    name string нет нет Название источника/группы/канала
    default_source number нет нет Источник по умолчанию
    source_channel_id number нет нет Канал
    source_group_id number нет нет Источник
    source_id number нет нет Источник детально
    name string нет нет Название источника/группы/канала
    is_custom boolean нет нет Является ли источник Пользовательским
    Настройки статического емайл трекинга.
    static_replacement_emails_settings array Может присутствовать в запросе только если tracking_type = static И rule_type = site_emails Настройка статических Email на сайте
    id number Идентификатор блока с настройкой.
    replacement_emails_groups array Группы email и сайтов для замены на подменяемые email.
    id number Идентификатор группы email. Метод получение списка существующих групп: "get.replacement_emails_groups"
    site_id number Идентификатор сайта.
    email_account object нет нет Адрес электронной почты
    id number нет нет ИД адреса электронной почты
    name string нет нет Название адреса электронной почты
    source object Источник, на которые настроен блок подмены.
    source_channel_id number Идентификатор канала источника. Метод получение списка существующих каналов: "get.get_sources_channels"
    source_group_id number Идентификатор группы источника. Метод получение списка существующих групп: "get.get_sources_groups"
    source_id number Идентификатор источника. Метод получение списка существующих Источников: "get_sources"
    name string нет нет Название источника или группы или канала
    default_source number Источник для отображения в отчете. Метод получение списка существующих Источников: "get.get_sources"
    Настройки динамического емайл трекинга.
    dynamic_replacement_emails_settings array Может присутствовать в запросе только если tracking_type = dynamic Настройка динамических Email.
    id number Идентификатор блока с настройкой.
    replacement_emails_groups array Группы email и сайтов для замены на подменяемые email.
    id number Идентификатор группы email. Метод получение списка существующих групп: "get.replacement_emails_groups"
    site_id number Идентификатор сайта.
    email_account object нет нет Адрес электронной почты
    id number нет нет ИД адреса электронной почты
    name string нет нет Название адреса электронной почты
    email_accounts array Доступно только если substitution_type = manual_pool нет нет Адреса электронных почт заведенных вручную
    coverage number нет нет Минимальный фактический охват по всем пулам ЕМТ
