    Список доступных методов

    Получение информации об аккаунте

    Метод Описание
    "getobj.account" Получение информации об аккаунте

    Отчеты пользователя

    Метод Описание
    "get.reports_list" Получение всех отчетов пользователя
    "create.report" Создание отчета пользователя
    "update.report" Редактирование отчета пользователя
    "delete.report" Удаление отчета пользователя

    Состояние отчета

    Метод Описание
    "getobj.report_state" Получение текущего состояния отчета
    "set.report_state" Изменение состояния отчета
    "create.shared_report_settings" Создание текущих настроек отчета с общим доступом

    Получение данных

    Метод Описание
    "getobj.report_table" Получение данных таблицы в отчете пользователя
    "getobj.report_total" Получение результирующих данных в отчете пользователя
    "getobj.report_transfer" Метод для перехода из агрегированного отчета в плоский отчет
    "get.communications" Получение списка всех обращений
    "get.deals" Получение списка всех сделок
    "get.leads" Получение списка всех лидов
    "get.visits" Получение списка всех посещений за последние 30 минут
    "get.chats" Получение списка чатов

    Выгрузка отчетов

    Метод Описание
    "create.report_generator_task" Создание задачи на выгрузку отчета файлом
    "get.report_generator_tasks" Получение списка задач на выгрузку отчетов файлом

    Глобальный календарь

    Метод Описание
    "get.report_global_calendars" Получение глобальных календарей пользователя
    "update.report_global_calendar" Редактирование глобального календаря

    Группы и типы отчетов

    Метод Описание
    "get.report_groups" Получение групп отчетов
    "get.report_types" Получение типов отчетов

    Фильтры

    Метод Описание
    "get.report_filters" Получение списка возможных условий фильтрации для отчета
    "get.report_rapid_filters" Получение списка возможных быстрых условий фильтрации для отчета

    Измерения

    Метод Описание
    "getobj.dimensions_tree" Получение вложенности измерений

    Пользовательские измерения

    Метод Описание
    "getobj.custom_dimension" Получение настройки пользовательского измерения
    "create.custom_dimension" Создание пользовательского измерения
    "update.custom_dimension" Редактирование пользовательского измерения
    "delete.custom_dimension" Удаление пользовательского измерения
    "get.custom_dimension_templates" Получение шаблонов для создания пользовательского измерения
    "get.report_entities" Получение списка доступных типов данных для использования в пользовательских вложенностях

    Метрики

    Метод Описание
    "get.metrics" Получение списка метрик
    "get.metric_filters" Получение списка доступных условий фильтрации для метрики

    Столбцы

    Метод Описание
    "getobj.columns_tree" Получение вложенности столбцов
    "delete.custom_column" Удаление пользовательского столбца
    "get.column_transfer_reports_list" Получение возможного списка отчетов для перехода по столбцу

    "update.custom_formula_column" Редактирование пользовательского столбца на основе формулы

    Пользовательские фильтры

    Метод Описание

    Цели

    Метод Описание

    Графики

    Метод Описание
    "getobj.report_linechart" Получение данных, агрегированных по времени и дополнительному измерениюй
    "getobj.report_linechart_total" Получение данных, агрегированных по времени суммарно
    "get.linechart_note_groups" Получение списка группированных заметок для отображения в линейном графике отчетов типа "Анализ рекламы"
    "create.linechart_note" Создание пользовательской заметки на графике
    "delete.linechart_note" Удаление пользовательской заметки

    Источники

    Метод Описание
    "get.sources" Получение списка Источников
    "get.source_filters" Получение списка фильтров для создания условий источников
    "getobj.source" Получение настройки Источника
    "create.source" Создание Источника
    "update.source" Редактирование Источника
    "update.source_state" Изменение статуса источника
    "update.priority_sources" Обновление приоритетов всех Источников
    "delete.source" Удаление Источника

    Каналы источников

    "update.source_channel" Редактирование канала Источника
    "update.source_channel_state" Изменение статуса канала источника
    "upload.source_costs" Загрузка расходов по Источникам
