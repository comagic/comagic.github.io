    Получение списка возможных условий фильтрации для источника

    Источники > Получение списка возможных условий фильтрации для источника
    Метод get.source_filters
    Описание Получение списка возможных условий фильтрации для источника
    Кому доступен Агент, Клиент
    Необходимые компоненты для работы с методом Data API для настройки
    Вес метода 1 балл

    Параметры запроса

    Название Тип Обязательный Описание
    access_token string да Ключ сессии аутентификации
    user_id number нет Уникальный идентификатор пользователя клиента агента от имени которого делается запрос
    Является обязательным для агента
    Для получения списка пользователей клиента необходимо использовать метод "get.customer_users"

    Параметры ответа

    Название Тип Допустимые значения Ответ по умолчанию Описание
    id enum да Уникальная мнемоника условия фильтрации
    name string да Название условия фильтрации
    description string да Описание условия фильтрации
    format enum да
    Операторы фильтрации
    operators array да Операторы фильтрации доступные для выбранного условия. Список всех операторов можно получить в разделе "Операторы фильтрации"
    id enum Мнемоника оператора фильтрации
    name string Название оператора фильтрации
    is_default boolean Оператор фильтрации по умолчанию
    Список возможных значений
    values array да Возвращается либо список возможных значений для выбранного условия, либо список Топ-10 значений из статистики за последние 7 дней

    JSON структура запроса

    {
    "jsonrpc":"2.0",
    "id":"number",
    "method":"get.source_filters",
    "params":{
        "access_token":"string",
        "user_id":"number"
    }
}
