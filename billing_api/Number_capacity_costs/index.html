    Стоимость номерных емкостей

    Получение списка стоимостей номерных емкостей

    Метод get.number_capacity_costs
    Описание Получение списка стоимостей номерных емкостей

    Параметры запроса

    Название Тип Обязательный Описание
    access_token string да Ключ аутентификации
    tp_id string да Уникальный идентификатор тарифного плана
    op_ids array of number нет Массив (возможно пустой) идентификаторов тарифных опций
    prefix string нет Префикс виртуального номера
    category string нет Категория (usual, bronze, silver, gold, platinum)
    owner_id string нет Уникальный идентификатор владельца номерной емкости
    number string нет Виртуальный номер (фильтры по prefix, category, owner_id будут игнорироваться)

    Параметры ответа

    Название Тип Допустимые значения Описание
    prefix string Префикс номерной емкости
    category string Категория номерной емкости
    location string Местонахождение номерной емкости
    owner_id number Уникальный идентификатор владельца номерной емкости
    user_charge number Абонентская плата
    activation_cost number Плата за подключение виртуального номера
    min_charge_sum number Минимальный платёж за виртуальный номер

    JSON структура запроса

    {
    "jsonrpc": "2.0",
    "id": 1,
    "method": "get.number_capacity_costs",
    "params": {
        "access_token": "1",
        "tp_id": 233
    }
}

    JSON структура ответа

    {
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "data": [
      {
        "prefix": "73012",
        "category": "usual",
        "location": "uud",
        "owner_id": 54,
        "user_charge": 350.0,
        "activation_cost": 250.0,
        "min_charge_sum": 1000.0
      },
      {
        "prefix": "73012",
        "category": "silver",
        "location": "uud",
        "owner_id": 54,
        "user_charge": 350.0,
        "activation_cost": 250.0,
        "min_charge_sum": 1000.0
      }
    ]
  }
}
