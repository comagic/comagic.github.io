    Шаблоны коммерческого предложения

    Получение списка шаблонов коммерческих предложений

    Метод get.cp_templates
    Описание Получение списка шаблонов коммерческих предложений

    Параметры запроса

    Название Тип Обязательный Описание
    access_token string да Ключ аутентификации
    project string да Проект (comagic, uis2, callgear)

    Параметры ответа

    Название Тип Допустимые значения Описание
    id number Уникальный идентификатор шаблонов коммерческого предложения
    name string Название шаблонов коммерческого предложения
    project string Название проекта
    document_layout string Html шаблон коммерческого предложения

    JSON структура запроса

    {
    "jsonrpc": "2.0",
    "id": 1,
    "method": "get.cp_templates",
    "params": {
        "access_token": "1",
        "project": "comagic"
    }
}

    JSON структура ответа

    {
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "data": [
      {
        "id": 4,
        "name": "Москва VIP",
        "project": "comagic",
        "document_layout": "<!DOCTYPE html>\n<html>\n <head>\n </head>\n <body>\n </body>\n</html>"
      },
      {
        "id": 3,
        "name": "СПБ",
        "project": "comagic",
        "document_layout": "<!DOCTYPE html>\n<html>\n <head>\n </head>\n <body>\n </body>\n</html>"
      }
    ],
    "metadata": {
      "version": null,
      "limits": {}
    }
  }
}
