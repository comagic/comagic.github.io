    Финальный расчет коммерческого предложения

    Получение финальной стоимости коммерческого предложения

    Метод get.profile_costs
    Описание Получение финальной стоимости коммерческого предложения

    Параметры запроса

    Название Тип Обязательный Описание
    access_token string да Ключ аутентификации
    tp_id string да Уникальный идентификатор тарифного плана
    op_ids array of number нет Массив (возможно пустой) идентификаторов тарифных опций
    components array of string нет Массив выбранных компонентов
    limits object нет Расширения лимитов
    number_capacity array of object нет Выбранные номерные емкости
    prefix string да Префикс номерной емкости
    owner_id number да Уникальный идентификатор владельца номерной емкости
    category string да Категория номеров в емкости, usual - обычный, bronze - бронзовый, silver - серебряный, gold - золотой
    count number да Количество номеров динамического КТ в блока с номерами на сайте
    component string да Мнемоническое название компонента

    Параметры ответа

    Название Тип Допустимые значения Описание
    Тарифный план
    tp array Тарифный план
    tp_id number Уникальный идентификатор тарифного плана
    cost number Объем списания
    service string Тип списания
    user_charge_tp - абонентская плата по тарифному плану
    Дополнительные опции
    ops array Опции
    op_id number Уникальный идентификатор опции
    cost number Объем списания
    service string Тип списания
    activation_op - плата за подключение опции
    user_charge_op - абонентская плата за опцию
    Лимиты
    limits array Лимиты
    limit string Мнемоническое название лимита
    tp_id number Уникальный идентификатор тарифного плана
    op_id number Уникальный идентификатор опции
    cost number Объем списания
    service string Тип списания
    user_charge_physical_limit - плата за расширение лимита
    Компоненты тарифного плана
    components array Компоненты
    component string Мнемоническое название компонента
    tp_id number Уникальный идентификатор тарифного плана
    op_id number Уникальный идентификатор опции
    cost number Объем списания
    service string Тип списания
    activation_component - плата за подключение компонента
    user_charge_component - абонентская плата за компонент
    Номерные емкости
    number_capacity array Номерные емкости
    prefix string Префикс номерной емкости
    category string Категория номеров в емкости
    usual - обычный
    bronze - бронзовый
    silver - серебряный
    gold - золотойplatinum - платиновыйpair - парный
    location string Мнемоническое название региона номерной емкости
    owner_id string Уникальный идентификатор владельца номерной емкости
    site string Адрес сайта
    ad_source string Рекламный источник
    site_block number Номер блока с номерами на сайте
    count number Количество номеров динамического КТ в блока с номерами на сайте
    tp_id number Уникальный идентификатор тарифного плана
    op_id number Уникальный идентификатор опции
    component string Мнемоническое название компонента
    cost number Объем списания
    service string Тип списания
    activation_number_capacity - плата за подключение номерной емкости
    user_charge_number_capacity - абонентская плата за пользование номерамиmin_charge_tp - минимальная абонентская плата по тарифу
